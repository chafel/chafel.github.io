Holy Grail Layout

在这一页中利用IFC布局后，再也不能使居中的文字位于图片正下方。遂研究table布局。发现table的强大（有别于inline-block）,虽有局限性（低级浏览器不支持dispaly:table-cell）但作者也说了，But if you only need to cater for mobile browsers, please feel free to flex away.此页面就是利用table实现的自适应,稳定且简洁。

读完鑫大的CSSer的学习瓶颈，深有感触啊。之前工作一个月后能布局各种页面效果了，还以为CSS不过如此，真是不知天高地厚。很多东西只是会用，不了解实现机制，细节把握不够深入。甚至看了一段时间JS荒废CSS后，连道面试题都搞不定。----页面仔们，扪心自问下：我每天几点下班？我每天下班后都干嘛了？我有专门花空余时间深入理解学习CSS的一些东西吗？我能够连续坚持数月吗？最后，我只说一句话：“吃得苦中苦，方为人上人”。深以为然，继续加油！
