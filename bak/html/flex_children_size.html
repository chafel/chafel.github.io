Flex content size 的标准 让浏览器对于 Flex 容器的的子元素最小尺寸的实现有所不同，因此对于不想让 flex 计算尺寸影响到的元素采用给定 min-height 或 min-width 或以下做法：

  .Flex-noAffect {
    flex-shrink: 0;
  }

对于不想让经 Flex 计算后的尺寸小于自身尺寸而又想当计算后尺寸超过自身尺寸后能适应到计算后尺寸的元素采用以下做法：

  .Flex-selfAdaption {
    // add 'px' to avoid a bug in IE 10-11
    // FYI:https://github.com/philipwalton/flexbugs#4-flex-shorthand-declarations-with-unitless-flex-basis-values-are-ignored
    flex-grow: 0px;
    flex-basis: auto;
  }

对于要允许计算后尺寸小于元素本身尺寸，需要出现滚动条的元素采用以下做法：

  .Flex-largeContent {
    flex: 1 0 auto;
  }
